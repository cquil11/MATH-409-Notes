• Binary operations:
(+): R × R 7→ R x, y ∈ R, x + y ∈ R,
(·): R × R 7→ R x, y ∈ R, x · y ∈ R.
• We have axioms for the real numbers as follows:
I Algebraic Axioms
(i) Associativity: For x, y, z ∈ R, we have
x + (y + z) = (x + y) + z
x · (y · z) = (x · y) · z

Bounded Intervals
• Given a < b, the set X of real numbers is a bounded interval if it is in the
form:
Open interval with endpoints a, b
{x ∈ X | a < x < b} = (a, b).
Closed intervals
{x ∈ X | a ≤ x ≤ b} = [a, b].
Half-open intervals
{x ∈ X | a < x ≤ b} = (a, b].
